Tauri/Rust의 도구·명령·문법·IPC·빌드 산출물을 한 페이지에. 인쇄용으로도 좋아요.

도구

rustupRust 버전 관리자.
cargo빌드·패키지 매니저 (Cargo.toml).
Tauri CLIscaffold / dev / build.
rust-analyzerIDE 자동완성 (LSP).
Vite프론트 번들러.

명령

npm create tauri-app@latest앱 뼈대 생성.
npm run tauri dev핫 리로드 개발.
npm run tauri build운영체제별 인스톨러.
cargo add <crate>Rust 의존성 추가.
