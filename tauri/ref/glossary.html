용어집 — 두려운 단어 사전

Tauri·Rust 용어마다 일상 비유 한 문장을 먼저, 그 다음 정의. 본문에서 data-term 용어에 hover하면 툴팁으로도 떠요.

Tauri

비유 — 웹으로 화면을 그리고, Rust로 일을 시키고, 운영체제의 창으로 띄우는 가벼운 데스크톱 앱 프레임워크.

공식: Rust 백엔드 + OS 네이티브 웹뷰 + 임의의 프론트엔드로 크로스플랫폼 앱을 만드는 프레임워크 (tauri.app).

Electron

비유 — 웹 기술로 데스크톱 앱을 만드는 older 형. Chromium을 앱마다 번들해 무겁다.

공식: Chromium + Node.js 번들 기반의 데스크톱 앱 프레임워크. GitHub 2013.

webview (웹뷰)

비유 — 운영체제가 기본 탑재한 웹 렌더러. Tauri가 Chromium 대신 빌려 쓰는 "식당 홀".

공식: WebView2(Win) / WKWebView(mac) / WebKitGTK(Linux) — OS 제공 웹 렌더링 컴포넌트.

Rust

비유 — 빠르고 메모리 안전한 시스템 언어. Tauri의 "주방" 언어.

공식: 소유권 기반 메모리 안전성을 가진 시스템 프로그래밍 언어. 가비지컬렉터 없이 안전 (rust-lang.org).

cargo

비유 — Rust의 빌드·패키지 매니저. npm 같은 존재. 의존성은 Cargo.toml.

공식: Rust 공식 빌드 시스템·패키지 매니저. Cargo.toml에 의존성을 선언.

crate (크레이트)

비유 — Rust의 패키지 단위. npm의 package 같은 것.

공식: Rust의 컴파일 단위/패키지. crates.io에서 배포.

rustup

비유 — Rust 버전 관리자. "Rust의 nvm".

공식: Rust 툴체인(컴파일러/cargo 등) 버전 관리자.

IPC (프로세스 간 통신)

비유 — Tauri에선 프론트↔Rust의 "주문서(invoke) ↔ 명령(command)" 통로.

공식: Inter-Process Communication. Tauri는 프론트엔드(JS)와 백엔드(Rust) 사이의 안전한 메시지 통신.

invoke()

비유 — 프론트가 Rust 명령을 부르는 "주문서". await로 결과를 받아요.

공식: @tauri-apps/api/core의 함수. 프론트엔드에서 Rust command를 비동기 호출.

#[tauri::command]

비유 — Rust 함수를 프론트에 노출시키는 표시(매크로). 주방의 "요리법".

공식: Rust 함수를 프론트엔드 invoke로 호출 가능하게 등록하는 속성 매크로.

macro (매크로)

비유 — 코드를 만들어내는 코드. "이 함수에 특별한 능력을 부여"하는 주문.

공식: 메타프로그래밍 장치. #[...] 속성 매크로, macro_rules! 등.

ownership (소유권)

비유 — "이 값은 누가 관리하나"를 정하는 Rust의 규칙. 본 가이드에선 깊이 다루지 않아요.

공식: 값의 수명과 해제 책임을 컴파일 타임에 정하는 Rust 핵심 규칙.

& (borrow, 빌림)

비유 — 값을 소유하지 않고 "빌려 읽는다"는 표시.

공식: 참조(reference). 소유권 이전 없이 값을 빌려 쓰는 것.

Result

비유 — "성공하거나 실패할 수 있음"을 나타내는 타입. 예외 대신 쓰는 안전한 방식.

공식: enum Result<T, E> = Ok(T) | Err(E). Rust의 표준 에러 처리 타입.

Option

비유 — "값이 있거나 없을 수 있음"을 나타내는 타입. null 대신.

공식: enum Option<T> = Some(T) | None. Rust의 null 안전 대체.

struct

비유 — 관련 데이터를 하나로 묶는 상자.

공식: 필드 모음으로 구조체를 정의하는 Rust 타입.

enum

비유 — "경우의 수"를 나열한 타입.

공식: 가능한 변형(variant)들을 나열하는 타입. Rust enum은 데이터를 가질 수 있음.

match

비유 — switch의 강력한 사촌.

공식: 패턴 매칭 제어문. 모든 경우를 빠짐없이 다뤄야 함.

let mut

비유 — 변수를 "바꿀 수 있게" 만드는 키워드. Rust는 기본이 불변이라 필요해요.

공식: 가변 변수 바인딩. 기본 let은 불변.

ACL (Tauri 2 권한)

비유 — Tauri 2의 최소권한 모델. 모든 API는 명시적 허용 필요. 기본은 차단.

공식: capabilities/ 설정으로 명령·API 접근을 명시 허용하는 권한 시스템.

bundle (번들)

비유 — 운영체제별 설치 파일. .msi/.dmg/.deb 같은 "포장 상자".

공식: cargo tauri build가 생성하는 플랫폼별 설치 패키지.

LSP

비유 — IDE에 코드 지능(자동완성/이동)을 제공하는 표준.

공식: Language Server Protocol. rust-analyzer가 Rust LSP 구현.
