T09 실전 미니 프로젝트

이름 입력 → Rust 명령 → 결과. 가장 작은 앱 회로 체험.
비유

Electron은 앱마다 브라우저를 한 벌씩 들고 다니는 겁니다. Tauri는 운영체제가 이미 깔아둔 창을 빌려 쓰고, 일은 Rust 주방에 맡깁니다. 그래서 설치 파일이 수 MB, 메모리도 가벼워요.
