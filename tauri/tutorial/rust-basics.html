Rust 대표 문법 7가지 — Tauri를 읽기 위해

Rust를 마스터하려는 게 아니에요. Tauri 예제를 읽을 수 있는 최소 7가지만. 이것만 알면 다음 페이지(IPC)가 술술 읽혀요.

비유

Rust는 안전한 주방 규칙이에요. "칼을 쓸 땐 이렇게", "재료는 이렇게 보관" 같은 규칙이 많아요. 하지만 Tauri 앱을 만들 때는 규칙 전부를 외울 필요 없어요 — 자주 쓰는 7가지만 알면 돼요.

조심하세요 — 범위 안내

본 가이드는 Rust를 가르치지 않아요. 소유권·수명·트레이트 같은 심층은 별도 Rust 입문 자료로 미뤄요. 여기선 7가지만 가볍게 훑고, 곧바로 Tauri에 씁니다.

1. fn — 함수

함수를 만드는 키워드. fn 이름(매개변수) -> 반환타입.

fn greet(name: &str) -> String {
    format!("안녕, {}!", name)
}

2. let / let mut — 변수 (기본이 불변)

Rust 변수는 기본이 불변(immutable). 바꾸려면 mut를 붙여요. "실수로 바꿨다"를 컴파일러가 막아줘요.

let count = 5;          // 불변 — 못 바꿈
let mut total = 0;       // 가변 — 바꿀 수 있음
total = total + 1;

3. match — 패턴 매칭

switch의 강력한 사촌. 가능한 경우를 빠짐없이 다뤄요.

match status {
    Ok(v)  => println!("성공: {}", v),
    Err(e) => println!("에러: {}", e),
}

4. struct / enum — 데이터와 경우의 수

struct는 관련 데이터를 묶고, enum은 "경우의 수"를 정의해요.

struct User { name: String, age: u32 }
enum Status { Win, Lose, Draw }

5. Result / Option — 실패·없음을 타입으로

Rust엔 null이나 예외 대신 타입을 써요. Option은 "있을 수도/없을 수도", Result는 "성공/실패"를 표현해요.

fn find(id: u32) -> Option<User> { ... }   // 없을 수도
fn load(path: &str) -> Result<Data, Error> { ... } // 실패할 수도

6. & — "빌려 쓴다" (소유권 냄새만)

&는 값을 빌려 쓴다는 뜻. &str은 "문자열을 소유하지 않고 빌려 읽는다"는 의미예요. 깊은 소유권 개념은 여기서 다루지 않아요 — "&가 붙으면 빌려 쓴다"만 기억하세요.

7. #[tauri::command] — Tauri 명령 (매크로)

Tauri가 프론트엔드에 노출할 Rust 함수에 붙이는 표시. 이게 있어야 프론트에서 invoke('greet')로 부를 수 있어요. (다음 페이지 T07에서 바로 씁니다.)

#[tauri::command]
fn greet(name: String) -> String {
    format!("안녕, {}!", name)
}
알아두면 좋은

#[] 모양은 매크로(macro)라 불려요. "이 함수에 특별한 능력을 부여"하는 주문 같은 거예요. #[tauri::command]는 "이 함수를 프론트에서 부를 수 있게 만들어 줘"라는 주문이에요.
