도구 생태계 — 주방 도구 정리

Tauri 개발에 필요한 도구 6가지. 모두 외울 필요 없어요 — 이름과 용도만 알면 검색할 수 있어요.

비유

주방을 상상해 보세요. rustup은 칼·프라이팬의 버전을 관리하는 도구함, cargo는 식자재 창고(npm 같은), Tauri CLI는 "Tauri 주방 세트"를 한 번에 깔아주는 키트. rust-analyzer는 주방장의 두뇌(자동완성)이에요.

설치 — 단 2단계

# 1. Rust 설치 (rustup — 버전 관리자)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 2. Tauri 앱 만들기 (이 명령 하나로 cargo·CLI·Vite 세팅까지)
npm create tauri-app@latest

사실 이게 전부예요. rustup으로 Rust를 깔고, npm create tauri-app으로 앱을 만들면 나머지 도구는 알아서 따라와요.

도구 6종 — 역할 한눈에

rustupRust 툴체인 관리자. "Rust의 nvm". 버전 설치/전환.
cargo빌드·패키지 매니저. "Rust의 npm". 의존성은 Cargo.toml.
Tauri CLI앱 스캐폴드/dev/build. cargo tauri dev / cargo tauri build.
rust-analyzerLSP(자동완성/타입/이동). IDE의 두뇌.
RustRover / VS CodeIDE. RustRover(JetBrains)가 정석, VS Code + rust-analyzer도 충분.
Vite프론트엔드 번들러. Tauri 공식 권장. create tauri-app에 포함.

Cargo.toml — Rust의 package.json

프론트엔드의 package.json처럼, Rust는 Cargo.toml에 의존성을 적어요. cargo가 이 파일을 읽고 알아서 다운로드·빌드해요.

[dependencies]
tauri = { version = "2", features = [] }
serde = { version = "1", features = ["derive"] }
팁

처음엔 VS Code + rust-analyzer면 충분해요(무료). Rust를 본격적으로 한다면 JetBrains RustRover가 정석이에요. 둘 다 rust-analyzer 기반이라 자동완성 품질이 비슷해요.
