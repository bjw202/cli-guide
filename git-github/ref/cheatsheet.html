참조 허브 · R04 · 인쇄용

한눈에 보는 치트시트

주제별로 한 장씩. 인쇄하려면 Ctrl+P (macOS: Cmd+P). A4에서 카드가 잘리지 않고 가독성을 유지해요.

기본

처음 시작하는 한 바퀴

git init빈 저장소 만들기
git clone <url>원격 저장소 복사
git config이름·이메일 설정
git status지금 상태 보기
git add <file>대기방으로 올리기
git commit -m "msg"사진첩에 한 장 붙이기
git log --oneline사진 목록(한 줄)
git show <커밋>사진 한 장 자세히

브랜치

평행 우주 만들고 합치기

git branch브랜치 목록/생성
git switch <브랜치>브랜치 이동 (-c 생성)
git merge <브랜치>두 우주 합치기
git diff <a> <b>두 갈래 차이
충돌 표시 제거<<</===/>>> 지우고 add

원격

세상과 주고받기

git remote add origin <url>원격 연결
git fetch받아만 오기
git pull받아서 합치기
git push -u origin main올리기(처음 연결)
git push --force-with-lease안전한 강제 push

복구

실수를 되돌리기

git reset --softHEAD만 뒤로
git reset --mixed대기방까지(기본)
git reset --hard작업방까지 버림
git revert <커밋>취소 커밋 추가(안전)
git restore <file>파일 이전 상태로
git clean -fd안 쫓는 파일 쓸기
git reflog숨은 발자취 찾기
git commit --amend직전 커밋 고치기

고급

역사 다루기

git rebase <브랜치>한 줄로 재정렬
git rebase -i대화형(squash/reword)
git cherry-pick <커밋>한 커밋만 가져오기
git stash작업 잠시 서랍행
git bisect이진 탐색 범인 찾기
git tag <이름>버전 이름표
git filter-repo역사에서 민감정보 제거
